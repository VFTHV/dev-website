Movies Entertainment App

This project was built with React front end framework and Redux state management library. I had mobile and desktop designs to work to and built it so that it was fully-responsive. I took a mobile-first approach and used Bootstrap CSS library for layout purposes.

Interaction Design / Front End Development React / Redux / SASS / Axios
Visit Website

In this project users are able to:

  • View the optimal layout for the app depending on their device's screen size
  • See hover states for all interactive elements on the page
  • Navigate between Home, Movies, TV Series, and Bookmarked Shows pages
  • Add/Remove bookmarks from all movies and TV series
  • Search for relevant shows on all pages

My workflow included following steps:

  • Identify semantic parts of the interface
  • Test and build API requesting patterns using Axios
  • Build Redux action creators and reducers
  • Apply API patterns with Redux
  • Identify parts of interface that could be isolated into separate React components
  • Build reusable React components
  • Apply components into appropriate interface parts
  • Add landmark aria and alt attributes
  • Refactor the code
