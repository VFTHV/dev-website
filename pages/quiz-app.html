Cards Game "Clubs Fool"

This is an imitation of a 3-player card game Based on Classic Russian-Ukrainian card game, this game was designed in raw JavaScript code (no libraries). It manipulated DOM elements to output the app onto an HTML canvas element. Relied on both event handling and the use of objects. At a later stage this project was partially transferred to Bootsrap 5 CSS library.

Interaction Design / Front End Development HTML / Bootstrap / DOM Manipulation
Visit Website

In this project users are able to:

  • View the optimal layout for each page depending on their device's screen size
  • Play with cards: beat weaker cards or take loer card in deck
  • Win when the player gets rid of all cards
  • Get winning message at the end of the game

My workflow included following steps:

  • Identify logics behind game flow
  • Build logics using functions for following: Recognition of stronger card Winning Condition Card deck shuffling and dealing Taking smaller card etc.
  • Add HTML markup
  • Refactor the code
