REST API Countries App

This project required me to build a responsive RESTful app that pulled data from a 3rd party server. It also had to have a color theme switcher. It is built on ReactJS framework. For routing I used React-Router-Dom library. I applied Redux state management library for storing app state data. For styling I used SCSS/SASS preprocessor. Axios API library was used for making GET requests.

Project Background

This project allowed me to extend my knowledge of Redux state management system. I also enhanced my skills with React HashRouter and BrowserRouter components. One of the tasks was to include Theme Color Switcher functionality. I have learned different ways to solve this problem as well. The provided design included mobile, tablet and desktop layouts.

My workflow included following steps:

  • Identify semantic parts of the interface
  • Test and build API requesting patterns using Axios
  • Build Redux action creators and reducers
  • Apply API patterns with Redux
  • Identify parts of interface that could be isolated into separate React components
  • Build reusable React components
  • Apply components into appropriate interface parts
  • Style the interface using SCSS preprocessor
  • Refactor the code
