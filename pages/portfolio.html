REST API Countries App

This project required me to build a responsive RESTful app that pulled data from a 3rd party server. It also had to have a color theme switcher. It is built on ReactJS framework. For routing I used React-Router-Dom library. I applied Redux state management library for storing app state data. For styling I used SCSS/SASS preprocessor. Axios API library was used for making GET requests.

IP Address Tracker App

This project required me to build a responsive IP Address and Domain tracker app. This app makes RESTful GET requests to 2 API servers: ipify.org and leafletjs.com. I did 2 variations of this project. On the first one I used vanilla HTML, SCSS and JavaScript. On the second one I used React, Redux, Axios.

This is an imitation of a 3-player card game Based on Classic Russian-Ukrainian card game, this game was designed in raw JavaScript code (no libraries).
